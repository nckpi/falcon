Welcome to our Self-Service Host Lookup Portal!

What does this tool do?

Provides an easy to use validation that CrowdStrike Falcon is installed and reporting in from your workstations and/or servers.

How does it work?

Simply enter your hostnames and/or cloud instance IDs in the box above, one per line, and click the Submit button to get your results.

How long should a search take?

Typically results are returned in seconds, but larger lists (hundreds or more) may take a few minutes. Maximum search results are capped at 5000 hosts, with duplicate name matches counting against against that limit. For example, searching for "test-system" might return "test-system1" and "test-system2" and count as two for the 5000 limit. Hitting the limit will prompt you to try with fewer names, so it is not left for you to guess. This is a vendor API limitation.

Does it only work for certain devices?

We can report on anything the Falcon sensor is installed on, as long as you have a hostname or instance ID. Servers, desktops, laptops, Windows, Linux, Mac, it is all here.

What if my system isn't found?

If the Falcon sensor is installed on a hosts and you cannot find it here, please reach out to the support team and we can work to determine why and fix it with you.

Are there any issues or catches?

  • Due to their nature and how they report their own names, AWS hostnames might not be found or may result in confusing duplicates from re-use of IP addresses. Please try to use AWS Instance IDs when available (e.g., i-0a3b12e2c12d12leo) to ensure unique matches.
  • Fully Qualified Domain Names (FQDNs) could be hit-or-miss so we implemented logic to automatically remove domains from hostnames you enter, saving you time in 'sanitizing' a spreadsheet of names. If you find that you have a match for a system but it uses domain names that don't make sense to you, separate verification is suggested.
